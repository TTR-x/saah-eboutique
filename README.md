# SAAH Business - Plateforme de Tontine Moderne

Bienvenue sur le projet SAAH Business. Cette application est une **PWA (Progressive Web App)** dédiée à l'épargne collaborative et à la gestion de plans de tontine.

## 🚀 Vision
Transformer l'épargne traditionnelle en une expérience numérique sécurisée, transparente et accessible partout, directement depuis votre écran d'accueil.

## 🎨 Design "Pro Simpliste"
Inspiré par les interfaces de **Facebook** et **WhatsApp**, le design privilégie :
- Un fond neutre et reposant (`#f0f2f5`).
- Des composants épurés (ShadCN UI).
- Une navigation fluide et intuitive.
- Une identité visuelle forte centrée sur le jaune primaire de SAAH.

## 🛠 Stack Technique
- **Frontend** : Next.js 15 (App Router), React 18, Tailwind CSS.
- **Composants** : ShadCN UI, Lucide Icons.
- **Backend** : Firebase (Auth, Firestore).
- **Mobile** : PWA (Manifest, Meta tags).
- **Images** : Cloudinary (pour le stockage dynamique).

## 📁 Organisation du Projet
- `src/app/` : Routes et pages de l'application.
- `src/app/admin/` : Espace de gestion réservé à l'administrateur.
- `src/components/` : Bibliothèque de composants réutilisables.
- `src/lib/` : Logique métier et services Firebase.
- `src/hooks/` : Hooks React personnalisés (Auth, Cart, Navigation).

## 📱 Installation PWA
L'application est configurée pour être installée sur Android et iOS via l'option "Ajouter à l'écran d'accueil" du navigateur.
